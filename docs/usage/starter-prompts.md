# Starter Prompts

These prompts are copy-ready entry points. Replace bracketed values with local
details and add one output mode: `quick triage`, `workplan`,
`audit/checklist`, or `artifact/template`.

## Strategy & Advisory

- Use `strategy-healthcare-consultant` in `workplan` mode: assess whether we should expand [service line] in [market], using [volume trend], [competitor data], [payer mix], [capital constraint], and [decision deadline].
- Use `strategy-operations-consultant` in `quick triage` mode: diagnose why [process KPI] worsened after [change], with [baseline], [current performance], [volume], [staffing], and [known constraints].
- Use `strategy-actuarial-advisor` in `audit/checklist` mode: review this [capitation/shared-risk] model for missing assumptions using [member months], [claims period], [risk scores], [trend], [stop-loss], and [credibility limits].

## Clinical Operations

- Use `clinical-utilization-management-specialist` in `quick triage` mode: evaluate this [observation/inpatient] status issue using [payer], [clinical facts], [order time], [criteria], [notice status], and [denial risk].
- Use `clinical-case-manager` in `workplan` mode: build a discharge barrier plan for [patient cohort/unit] using [EDD], [post-acute need], [payer auth status], [family preference], and [placement barriers].
- Use `clinical-prior-authorization-specialist` in `artifact/template` mode: draft a prior authorization appeal packet for [service] denied by [payer], using [policy criterion], [denial reason], [clinical evidence], and [deadline].

## Quality, Safety & Compliance

- Use `quality-accreditation-specialist` in `audit/checklist` mode: prepare survey readiness for [setting] against [accreditor/standard], using [evidence files], [prior finding], [owner], and [survey date].
- Use `quality-compliance-officer` in `quick triage` mode: assess this potential [HIPAA/Stark/AKS/FCA/EMTALA] issue using [facts], [dates], [parties], [payment/PHI flow], [policies], and [known advice].
- Use `quality-patient-safety-officer` in `artifact/template` mode: draft an RCA2 outline for [event], using [timeline], [harm level], [immediate containment], [policies], and [known contributing factors].

## Revenue Cycle & Finance

- Use `revenue-cycle-specialist` in `quick triage` mode: diagnose why clean claim rate dropped for [payer/service line] using [date range], [claim volume], [dollars], [CARC/RARC mix], and [workflow changes].
- Use `revenue-contract-analyst` in `artifact/template` mode: produce a contract performance scorecard for [payer/product], using [contract terms], [claims/remits], [fee schedule], [effective dates], and [underpayment examples].
- Use `revenue-finance-manager` in `workplan` mode: build a service-line margin review for [service line], using [volume], [payer mix], [direct costs], [allocation method], [capital needs], and [decision deadline].

## Payer & Managed Care

- Use `payer-value-based-care-manager` in `workplan` mode: prepare downside-risk readiness for [contract/model], using [attributed lives], [baseline cost], [quality gates], [care-management capacity], and [performance period].
- Use `payer-credentialing-enrollment-coordinator` in `audit/checklist` mode: review enrollment readiness for [provider group], using [roster], [locations], [NPIs/TINs], [CAQH/PECOS status], [payer list], and [target effective date].
- Use `payer-relations-specialist` in `quick triage` mode: evaluate our leverage in a payer dispute with [payer/product], using [contract clause], [claim examples], [timeline], [market facts], and [desired outcome].

## Health IT & Informatics

- Use `healthit-interoperability-engineer` in `quick triage` mode: troubleshoot this [HL7/FHIR/X12] failure using [source system], [target system], [payload/error], [workflow trigger], [volume], and [auth/security context].
- Use `healthit-informatics-manager` in `workplan` mode: design governance for [CDS/data standard/EHR optimization] using [affected users], [patient-safety or regulatory driver], [maintenance owner], and [success metric].
- Use `healthit-clinical-data-analyst` in `artifact/template` mode: draft a dashboard specification for [quality measure/registry], using [cohort], [numerator/denominator], [data sources], [refresh cadence], and [audience].

## Operations & Administration

- Use `operations-hospital-administrator` in `quick triage` mode: diagnose ED boarding despite open beds using [census], [bed board], [discharge forecast], [staffing constraints], [bottleneck service], and [target KPI].
- Use `operations-ambulatory-manager` in `workplan` mode: improve third-next-available appointment for [clinic], using [provider templates], [demand/backlog], [no-show rate], [room count], and [staffing model].
- Use `operations-workforce-manager` in `audit/checklist` mode: review agency labor reduction options using [census/acuity], [staffing grid], [vacancies], [overtime], [agency spend], and [labor constraints].

## Population Health & Community

- Use `pophealth-population-health-manager` in `workplan` mode: create a care-gap closure campaign for [measure/cohort], using [attributed population], [gap file], [outreach capacity], [SDOH data], and [contract goal].
- Use `pophealth-community-health-coordinator` in `artifact/template` mode: draft a CHNA implementation plan for [community need], using [geography], [partners], [budget], [grant/reporting terms], and [metrics].
- Use `pophealth-surveillance-coordinator` in `quick triage` mode: assess whether [case cluster] requires reporting or outbreak activation using [jurisdiction], [case definition year], [line list], [lab results], and [exposure timeline].

## Pharmacy & Drug Programs

- Use `pharmacy-benefits-specialist` in `artifact/template` mode: build a PBM contract scorecard for [contract], using [drug spend], [rebates], [specialty terms], [formulary tiers], and [member impact].
- Use `pharmacy-medication-safety-specialist` in `quick triage` mode: evaluate a medication safety event involving [drug/process], using [harm level], [MAR/order/pump data], [workflow step], and [current safeguards].
- Use `revenue-340b-program-manager` in `audit/checklist` mode: review 340B duplicate-discount risk for [entity/site], using [Medicaid IDs], [contract pharmacy setup], [accumulator sample], and [audit period].

## Emergency Preparedness

- Use `emergency-preparedness-coordinator` in `workplan` mode: update the emergency operations plan for [hazard], using [HVA score], [affected services], [resource constraints], [HICS roles], and [exercise deadline].
- Use `emergency-preparedness-coordinator` in `artifact/template` mode: draft an after-action report for [exercise/event], using [objectives], [timeline], [participants], [capability gaps], and [corrective-action owners].
- Use `emergency-preparedness-coordinator` in `quick triage` mode: determine immediate activation needs for [incident], using [facility impact], [patient care constraints], [staffing], [supplies], and [external partners].
